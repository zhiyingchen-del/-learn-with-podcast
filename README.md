# Learn with Podcast 🎙️

用 Podcast 學英文和法文的語言學習工具。

## 功能
- RSS Feed / 音訊連結匯入
- Whisper API 自動生成逐字稿
- 點擊單字即時查詢（Claude API）
- 字卡庫（自動儲存於瀏覽器）
- AI 動態出題測驗
- 翻牌複習模式

## 使用方式
1. 開啟 app 後前往「設定」
2. 輸入 Claude API Key（取自 console.anthropic.com）
3. 輸入 OpenAI API Key（取自 platform.openai.com，Whisper 用）
4. 到「Podcast 集數」貼入 RSS Feed 網址開始使用
